## Installation

Card is already installed in `tamagui`, or you can install it independently:

<CodeBlockWithTabs>
  <div id="yarn">
  ```bash margin=false
  yarn add @tamagui/card
  ```
  </div>

  <div id="npm">

  </div>

## Anatomy

```tsx
import { Card } from 'tamagui' // or '@tamagui/card'

export default () => (
  <Card>
    <Card.Header />
    <Card.Footer />
    {/* any other components */}
    <Card.Background />
  </Card>
)
```

## API Reference

### Card

Frame of the card, extends [ThemeableStack props](/docs/components/stacks#themeablestack), adding:

<PropsTable
  data={[
    {
      name: 'size',

    {
      name: 'unstyled',
      required: false,
